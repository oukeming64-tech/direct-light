# 更新记录 · Changelog

本文件记录面向用户的版本变化。详细的协作过程和逐版分工见 [`COLLABORATION.md`](COLLABORATION.md)。

格式参考 [Keep a Changelog](https://keepachangelog.com/)，版本号为产品内部迭代号（非 npm 语义化版本）。

## [0.7.0] - 2026-06-21 · 可开源第一版

可开源第一版收口。功能与 v0.6e 一致，新增开源工程化收口：

- 新增 `LICENSE`（MIT）、`CONTRIBUTING.md`、`CHANGELOG.md`、英文 `README.en.md`。
- `package.json` 补齐开源元数据（描述、license、author、repository、keywords、`engines`），版本号 `0.7.0`。
- README 顶部新增「快速开始 / 项目结构 / 已知限制 / 许可证」开源门面，原产品需求文档保留在下方。
- 清理仓库杂物，规范 `.gitignore`，初始化 git 仓库并开源到公开仓库。
- 新增 GitHub Actions 自动部署：每次 push 到 `main` 自动构建并发布在线 demo —— https://oukeming64-tech.github.io/direct-light/ （`vite.config.ts` 生产构建使用 `base: '/direct-light/'`，开发仍在根路径）。
- 顶栏版本号 `v0.6e → v0.7.0`。

已知限制：最多 3 盏灯、UI 仅简体中文、桌面优先、渲染为沟通向近似（见 README）。

## [0.6.0] - 2026-06-21 · 控光线（v0.6a–v0.6e）

灯上控光附件与棚内独立控光器材全线完成。

- **v0.6a** 灯上附件 MVP：中号柔光箱 / 蜂巢 / 标准反光罩 / 柔光布，通过 `modifierId` 影响有效亮度、光束、柔硬和彩色溢光，不破坏原始灯光参数。
- **v0.6b** 附件 3D 可见形体 + 镜头视角「导演简介」状态条（如 `Key Light · Nanlux Evoke 600C · 中号柔光箱 · 柔光主光`）。
- **v0.6c** 棚内独立控光器材作为场景对象：黑旗 / 反光板 / 柔光布框，可拖动、旋转、改尺寸（本阶段零光学）。
- **v0.6d** 控光器材近似光学：黑旗吃光 / 负补光 / 简化遮挡、反光板暗部弱补光、柔光布框在灯人之间软化；运行时派生，不写回灯光或器材数据。
- **v0.6e** 收口：A/B 差异摘要把「控光器材」从「道具」拆出；保存 / 加载 / 复制 / A-B 不丢附件与派生光学；导出图保留器材视觉。

## [0.5.0] - 2026-06-20 · 灯具预设库 + 渲染可信度

- **v0.5.0** 8 个灯具器械预设（COB 白 / 双色温、Nanlux Evoke 600C、LED 面板、RGB 灯管、菲涅尔、小型效果灯、暖色实景灯）；面板「器械」下拉一键套用默认光质，仍可手动微调；新增能力标签（全彩 / 双色温 / 暖色 / 白光）。
- **v0.5.1** 渲染可信度：一体化无缝弧形背景、可见灯体、地面反射进入填光与彩色反弹公式。

## [0.4.0] - 2026-06-19 · 人物姿态 + 摄影机控制

- **v0.4a–b** 层级骨架假人、9 个姿态预设、关键关节滑杆、坐姿与承载面坐/站分流、前臂内旋。
- **v0.4.5** attach-to-support：人物绑定到桌椅 / 小舞台，承载物移动旋转时人物跟随。
- **v0.4.6** A/B 对比产品引导：差异摘要、B 冻结来源/时间、空状态三动作。
- **v0.4.7** 摄影机控制：方位角 / 距离 / 高度 / 看向高度 / 焦段 / 画幅、5 个机位预设、从自由视角取景、目标锁定（手动 / 锁定人物 / 多人中心）、棚内夹紧。
- **v0.4.8** `store.ts` 无行为变更拆分（685 → 38 行，八组 action 工厂）。
