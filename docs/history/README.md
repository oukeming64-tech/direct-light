# Direct Light History Archive

This folder keeps completed or superseded project history out of the root docs.

## Folders

- `specs/` — completed implementation specs such as v0.4 camera, v0.6 gear, v0.8 multi-light, v0.9 custom fixtures, and v0.10 i18n.
- `handoffs/` — old one-off handoff documents.
- `snapshots/` — full historical versions of long root docs before the 2026-06-29 cleanup.

## Standalone History Files

- `COLLABORATION_HISTORY_2026-06-23.md` — long collaboration and release narrative through the v0.9 / v0.10 transition.
- `HERMES_HISTORY_2026-06-23.md` — old Hermes-oriented history.

## Cleanup Scope

Changed on 2026-06-29: old specs and long root documents were moved here, while root docs were rebuilt as short current-entry files.

Not changed: app code, showcase code, build config, package metadata, release versions, tags, deployment workflows, public URLs, or product behavior.
